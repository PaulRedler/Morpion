L'activité est à faire à 2. Le but étant d'ajouter une fonctionalité de chat en ligne en observant les fichiers fournies.
Étapes pour lancer l’activité

Choisir un ordinateur qui servira de serveur.

Ouvrir le terminal dans le dossier contenant server.js.

npm init -y puis npm i ws 

Lancer le serveur : node server.js

Dans le code HTML, modifier l’adresse du serveur WebSocket du fichier  morpionenligne a la ligne 67.
